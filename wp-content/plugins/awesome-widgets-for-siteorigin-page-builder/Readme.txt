= Overview =

**Awesome Widgets for SiteOrigin Page Builder** is a great plugin and it help you add Woocommerce Products Featured Collection, New Arrival, Sale Off, Best Seller and Category on any page of your site.

= Free Features: =

* <strong>NEW</strong> The plugin is compatible with our <strong>Support Ticket System By Phoeniixx plugin</strong>.

**Featured Collection Widget**
Any product that has been set as 'Featured' would appear under this widget. 
You could set the widget on any page of your website.

**New Arrival Widget**
The recently added products would be displayed under this widget. 
You could set the widget on any page of your website.

**Sale OFF Widget**
All the On-Sale products would be displayed under the Sale Off Widget. 
The widget could be added on any page.

**Best Seller Widget**
Products that have been sold maximum number of times are automatically picked and displayed under this widget. 
The widget could be added on any page.

**Category Widget**
Same Category Products are displayed under Category Widget. 
The widget could be added on any page.


**Common Features of Above Widgets**
a.	You could edit the Title of the widget
b.	You could set the number of products that you want to display at a time.
c.	You could set the carousel as 'on' or 'off'

== Installation ==

1. Install [WordPress](https://wordpress.org/)
2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.
3. For the plugin to work as it should, WooCommerce plugin has to be installed and enabled.
